Στο assignment χρησιμοποιόυμε το lampp ( Linuxs version του xampp ) καθώς δουλεύουμε στο WSL περιβάλλον.

---

To ξεκινάμε στο bash στο directory που έχει εγκατασταθεί το lampp με: sudo /opt/lampp/lampp start

και αντίστοιχα σταματάμε με: sudo /opt/lampp/lampp stop

---

Το πρόγραμμα ξεκινάει πηγαίνοντας στο root του Project όπου τρέχουμε στο bash: node app.js

---

Χρειάζεται δημιουργία access token από την ιστοσελίδα Hugging face για την χρήση του AI / LLM και η αποθήκευση του στο root του κώδικα του project μέσα στο αρχέιο .env με την εξής εισχώρηση:

HF_TOKEN=<προσωπικό_access_token>

όπου <προσωπικό_access_token> μπαίνει το value του Access Token.
